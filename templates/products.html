In the era of big data, data analytics and visualization are cutting-edge technologies with data as its essence. To analyze and visualize high-dimensional datasets, DaGoo, a versatile platform for big data, comes out. DaGoo is skilled at doing fast analysis for any data of high dimension and can figure out series of problems related to big data and artificial intelligence. Getting a detailed report with one click right after uploading data, users are not required to code. To establish academic atmosphere, users can exploit, compare and share their methods on the platform. For more details, see the Instructions!

DaGoo has four inseparable brothers: TableGoo, TextGoo, GeoGoo and GraphGoo. They are geniuses in different areas!


TableGoo! —— good at table data analysis.
Imported data type: table data
Table format for unloaped files:
First lines: attributes; First column: identifiers; The rest: data
Functions with table data:
Analysis and Visual Charts

TextGoo! —— skilled in text data analysis.
Imported data type: text data
Table format for unloaped files:
pictures or input in textbox(Word Cloud)
First column: source; Second column: target; Third column: relationship(Knowledge Graph)
Functions with text data:
Word Cloud and Knowledge Graph


GeoGoo! —— adept at geographical data analysis.
Imported data type: geographical data
Table format for unloaped files:
First lines: attributes; First column: areas; The rest: data
Functions with geographical data:
Chinese map with various attributes


GraphGoo! —— specialize in graph data analysis.
Imported data type: graph data
Table format for unloaped files:
First lines: attributes; First column: identifiers; The rest: data
Functions with graph data:
social networks and recommendation system
